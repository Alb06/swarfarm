Personal Logs

You can access the logs you have contributed and view personal stats. Your logs are available to download in CSV format via the links on the left.

How to get personal logs

  1. Download the latest package from the SWProxy-plugins project on Github.
  2. One time only - import your game data via SWProxy export
  3. Make sure you have a SwarfarmLogger.py plugin in the SWProxy plugins directory.
  4. Play the game while connected to the proxy.
