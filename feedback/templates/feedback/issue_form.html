Important!

Read the following guidelines before submitting feedback. More information leads to better results and saves time for everyone.

Monster, skill or other game data corrections

Include what you think should be the correct information. Do not just point out that something is wrong. Try to point out all the skills/monsters that need correcting, as many monsters share skills. Provide evidence for your change if it is not an obvious error. Other fan-created websites and the Summoner's War Wiki are not evidence. In-game screenshots, videos, or data mined information are evidence.

Profile import errors

Only SW-Exporter's profile JSON file is supported for importing. Read the import instructions and follow them precisely. Only submit feedback if you are sure you did everything correctly.

Did you get an error like 'xxxxxx' is a required property? Profile data goes through strict validation to ensure it is correct before processing. Your uploaded file is missing required information and can't be processed. This can happen randomly due to Com2US being inconsistent or trying to import files from unsupported applications. If you are using SW-Exporter, try logging in and out of the game and importing again.

New features or improvements

Provide detailed information on how you think the feature should work and why it is a useful thing to have. If you are interested in coding, SWARFARM is open source and I will gladly accept code contributions. Please discuss features and implementation in a GitHub issue before spending a lot of time working on something.

General website issues

Describe what is wrong and what should be correct. Provide repeatable steps to reliably create the error. Include screenshots or links of the issue, with notes or markup if it's helpful. Images can be uploaded to imgur or other image hosting service and linked in your description.
