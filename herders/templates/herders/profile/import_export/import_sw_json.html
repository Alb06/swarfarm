Import Summoner's War JSON

Instructions:

  1. Download the latest release of the SW-Exporter program.
  2. Follow the instructions provided to capture your Summoner's War account data.
  3. Find the acctname-1234567.json file and upload it here. The name and numbers in the filename will be different.
  4. Optional: You can import a friend visit data file, which is named 'friendname-1234567-visit.json'
    • You might want to do this if you can't run SW-Exporter yourself, but have a friend who can.
    • This data will be imported into the currently logged in SWARFARM account.
    • This will not import unequipped rune inventory! This data is only available to the actual account owner, not friends.
    • If you already have runes in your SWARFARM profile inventory, you can choose not to delete them.

For your first upload, it is recommended to clear your profile or you will end up with a lot of duplicates. For subsequent uploads your account will by synced automatically. Please report any issues by leaving feedback here. You can also email your .json file to swarfarm@porksmash.com so I can investigate specific issues.
